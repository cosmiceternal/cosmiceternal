# Microsoft's own telemetry-endpoint documentation and in open-source debloat
# projects.

# NOTE: This list is intentionally CONSERVATIVE. It targets pure telemetry
# collectors and deliberately EXCLUDES anything shared with Windows Update,
# the Microsoft Store, licensing, or time sync, because blocking those breaks
# legitimate functionality. Blocking is done by routing the name to 0.0.0.0.
